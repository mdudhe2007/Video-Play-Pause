<h1>🎬 Video Player Web App</h1>

### 📌 Overview

A simple and responsive video player built using HTML, CSS, and JavaScript.
This project demonstrates basic media control functionality including play and pause actions with a styled user interface.

---

### 🚀 Features

▶️ Play video<br>
⏸️ Pause video<br>
🔁 Loop playback<br>
🎨 Styled UI with Orbitron font<br>
📱 Responsive design<br>
🌌 Custom background support<br>

### ⚙️ How It Works

*The 'video' element loads and displays the media file.<br>

*JavaScript controls playback using:<br>

*video.play()<br>
*video.pause()<br>

*Buttons trigger these functions via event listeners.<br>

---
### 🧠 Key Concepts Used

*DOM Manipulation<br>

*Event Handling<br>

*Media API (HTML5 Video)<br>

*CSS Styling & Layout<br>

### 📥 Setup & Usage

Clone the repository:<br>
```
git clone https://github.com/mdudhe2007/Video-Play-Pause.git <br>
```
Open the project folder<br>

Open `index.html` in your browser<br>

---

### 📌 Notes
*Make sure the video file path is correct<br>

*Keep all assets in the same directory or update paths accordingly<br>

*Works best in modern browsers<br>
